# Strategy Method - SimUPato

### SimUPato project uploaded using the Strategy method.

This repository contains the complete NetBeans project for SimUPato, demonstrating the implementation of the Strategy design pattern in Java -.-

## Repository Details

* **IDE:** NetBeans
* **Design Pattern:** Strategy
